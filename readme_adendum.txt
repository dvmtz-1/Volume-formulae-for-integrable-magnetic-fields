Codes implementing Theorem 1 and 4 from △[M24] to integrable vector fields in toroidal geometry:
- Example 1: Axisymmetric field in cilindrical coordinates from ◭[BKM20
- Example 2: Toroidal helical magnetic field introduced in ◉, based on (Kallinikos et. al 2014◧)

The results from this implementation are presented in ◪ [MM25b]

'V_Ex1_Eq1.py' : Implements the volume formula in Eq(1) to Example 1.
'V_Ex1_EqA1.py': Implements the volume formula in Eq(1) to Example 1.
'V_Ex1_Thm1' :   Implements the volume formula in Thm1 to Example 1.
'ini_par_Ex1.txt' : Parameter values file for Example 1 scripts.

'V_Ex2_Eq1.py' : Implements the volume formula in Eq(1) to Example 2.
'V_Ex2_EqA1.py': Implements the volume formula in Eq(1) to Example 2.
'V_Ex1_Thm3p.py' : Implements the volume formula in Thm3' to Example 2.
'V_Ex1_Thm4.py' :  Implements the volume formula in Thm4 to Example 2.
'ini_par_Ex2.txt' : Parameter values file for Example 2 scripts.
'V_Ex2_utilities.py' : Vectorial, numeric and utility functions used in Example 2 scripts.
